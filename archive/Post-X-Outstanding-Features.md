# Post-X Outstanding Features

Phase X has concluded, and we deem it a great success.  The game feels cohesive and appropriately challenging.  It is starting to develop a proper identity, and for the first time, we can start to see on the horizon a point where the primary development bottleneck will be content authoring rather than feature development.  However, there is still much to add.  To name a few:

* A currency and shop system
* A reward system
* The much-deferred unit rarity and drafting system
* The much-deferred event system.
* The deferred starting character system

  * Different starting rosters and daemons
  * Possibly different unit draft pools
* Consumable items
* Neutral units

  * Not to be confused with our current walls and half-cover units.  Proper encampments of neutrals that can be fought for additional rewards, ala Warcraft 3.
* Dynamic terrain

  * This will probably come in two flavors: destructible "terrain," which will be like our current, static neutral units, and terrain tiles with dynamic effects.
* Additional terrain tile types:

  * Deep water
  * Uneven terrain / "mountains"
  * Mud
  * Ice
  * Sand
* Flight
* Multi-tile units
* Non-stat-based unit effects

  * Burn
  * Bleed
  * Poison
  * Blind
  * Confusion
  * Panic
  * Frozen
* Additional attack mechanics

  * Chain attacks
  * Attacks that inflict status effects
  * Summoning
* Difficulty and meta progression

Additional dev tools must also be built:

* The archetype editor must be expanded to allow the creation of new archetypes
* We need an attack editor that allows the (reasonably) dynamic creation of new attacks
* We'll need a draft pool editor 
* Also an event editor 
* Probably a starting character editor, though that might be simple enough to be doable via JSON

Additional polish:

* Additional SFX
* A way to represent status effects in-battle
* Camera shake
* A way to actually ship the game
